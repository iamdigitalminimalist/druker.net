### Create a UI with Vanilla JavaScript and DOM

```js
<body>
  <div id="root"></div>
  <script type="text/javascript">
    const rootElement = document.getElementById('root') const element =
    document.createElement('div') element.textContent = 'Hello World'
    element.className = 'container' rootElement.appendChild(element)
  </script>
</body>
```
